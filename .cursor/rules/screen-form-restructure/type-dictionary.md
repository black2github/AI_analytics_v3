# Словарь типов полей (UI-типы ЭФ)

Колонка **«Тип»** в таблице полей заполняется **UI-типом элемента экранной формы**
— так же, как это принято в исходных описаниях ЭФ. Отдельный словарь типов
данных/форматов значений в проекте пока не ведётся; при необходимости расширим.

## Базовый набор (закрытый, расширяемый)

- **Текст** — однострочный/многострочный ввод.
- **Число** — числовой ввод.
- **Пароль** (Password) — маскируемый ввод.
- **Поиск** (Search) — ввод с контекстным поиском/подсказками.
- **Дата** (Date) — выбор даты (календарь).
- **Список** (Dropdown / Select) — выбор одного значения из списка.
- **Список с множественным выбором** (Multi-select) — выбор нескольких значений.
- **Переключатель** — набор взаимоисключающих значений (radio / toggle).
- **Чекбокс** — бинарный флаг.
- **Блок загрузки** — область загрузки файла (drag-and-drop).
- **Гиперссылка** — ссылка/кнопка-ссылка (в т.ч. Button-link, кнопка скачивания).
- **Заголовок** — служебная строка-заголовок раздела/блока (не поле ввода).

## Приведение нестандартных типов исходника

Типы из выгрузки Confluence приводи к ближайшему каноническому, сохраняя смысл:

| В исходнике | Канонический тип |
|-------------|------------------|
| «Иконки/Текст», «Иконка статуса + Текст» | Заголовок (декоративный, если не поле ввода) |
| «Текст + гиперссылки» | Гиперссылка (если ядро — ссылка) или Текст |
| «Кнопка-ссылка на скачивание (Button-link)» | Гиперссылка |

Если встречается **действительно новый** UI-тип, которого нет в наборе — добавь его
в этот список (не выдумывай разовую нотацию молча) и используй.

## Кандидаты в словарь форматов значений (опционально)

Повторяющиеся/общесистемные **форматы значений** (ИНН — 12 цифр, БИК — 9 цифр,
номер счёта — 20 цифр, код валюты — 3 буквы) можно собрать отдельным
списком-предложением на решение аналитика — как заготовку под будущий словарь
форматов. Это **не** колонка «Тип» (там UI-тип); формат значения живёт в колонке
«Формат». Словарь форматов самовольно не создаётся — только кандидаты.

```
| Кандидат формата | Ограничение | Поля-источники |
|------------------|-------------|----------------|
| Счёт.Номер | ровно 20 цифр | Номер счёта, Счёт списания комиссии |
| Код.Валюта | 3 заглавные буквы | Валюта |
```
