# Именование и размещение (общие с control-split)

- **Имя файла — из оригинального имени минимальной правкой**, без пробелов
  (дефисы), `.md`, URL-safe. Не пересобирать из «логического» названия.
- **Маркер `SFR` (Screen Form Restructured)** — результат переформатирования
  **всегда** выносится в **отдельные** файлы с маркером `SFR` (см. правила ниже).
  Исходный файл **не изменять и не удалять** — проверяющий аналитик заменит его
  вручную после ревью, если сочтёт нужным. Режим автоматического замещения
  исходника скиллом **не применяется** (может быть добавлен отдельно позже).
  - **Есть префикс в квадратных скобках** — `SFR` **сразу после префикса**
    (чтобы SFR-файлы были видны в начале длинного имени):
    - ЭФ: из `[КК_ВК]-ЭФ-Клиента-страница-Анкеты.md` получается
      `[КК_ВК]-SFR-ЭФ-Клиента-страница-Анкеты.md`
    - FE-контроли: из `[КК_ВК]-ЭФ-Клиента-страница-Анкеты.md` получается
      `[КК_ВК]-SFR-FE-Контроли-ЭФ-Клиента-страница-Анкеты.md`
  - **Нет префикса в квадратных скобках** — `SFR-` **первой частью** имени файла:
    - ЭФ: из `ЭФ-Заявка-на-открытие-счета.md` получается
      `SFR-ЭФ-Заявка-на-открытие-счета.md`
    - FE-контроли: из `ЭФ-Заявка-на-открытие-счета.md` получается
      `SFR-FE-Контроли-ЭФ-Заявка-на-открытие-счета.md`
- Кандидаты в форматы значений (опционально): `<имя-ЭФ>-Кандидаты-в-форматы.md`
  либо раздел в конце ЭФ (по объёму). Колонка «Тип» формы — UI-типы, не форматы
  (@.cursor/rules/_shared/type-dictionary.md).
- **Реакции не именуются и не создаются здесь.** Информирование об ошибках ведётся
  единым документом сервиса (для КК — `[КК] Правила информирования пользователя
  Клиента об ошибках` в `[КК]-Библиотека-модулей-Сервиса/`); ссылайся на него,
  не дублируй.
- **`title` и `doc_id` согласованы с именем файла** (та же основа; `title` —
  пробелы, `doc_id` — `{{<service>: <текст title>}}`). Рассинхрон ломает
  смарт-ссылки.
- **`related`** — строка со смарт-ссылками через запятую (не YAML-массив): ЭФ ↔
  FE-контроли двусторонне.
- Квадратные скобки в именах: артефакт выгрузки `_КК_ВК_` заменяется на каноничное `[КК_ВК]`.
- **Каталог**: выходные файлы — рядом с обрабатываемым оригиналом.
