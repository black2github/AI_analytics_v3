## 🗺️ План интеграции (с номерами артефактов)

### Шаг 1: Подготовка
```bash
mkdir -p app/agents
touch app/agents/__init__.py
```

### Шаг 2: Копирование файлов

1. **Артефакт #1** → `app/agents/agent_config.py`
2. **Артефакт #12** → `app/agents/agent_tools.py` (переименовать!)
3. **Артефакт #17** → `app/agents/requirements_agent.py` (LangGraph версия!)
4. **Артефакт #4** → `app/routes/agent.py`

### Шаг 3: Регистрация
Используйте пример из **Артефакт #11** (main.py)

### Шаг 4: Следуйте чек-листу
Откройте **Артефакт #18** (LANGGRAPH_MIGRATION.md) для полной инструкции
