# Заполнение справочника

- 1 - код бизнес сервиса
- 2 - код микросервиса – используется в коде и заполняется разработчиками. Приведен справочно.

Для отличия "Кода сервиса" и "Кода микросервиса" добавляется префикс "BS" в код сервиса, для указания, что это код бизнес-сервиса.
Например: Сервис лизинга
- Код сервиса = LEASING_**BS**
- Код микросервиса = LEASING
